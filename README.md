# APEX-Testnet-Node
Non validator node setup script for the community testnet

# OPS
Ubuntu

# How-to
Clone the repo:

> git clone https://github.com/yuomii/APEX-Testnet-Node.git

change dir:

> cd APEX-Testnet-Node 

Make both scripts executable:

> chmod +x install.sh

> chmod +x run.sh

Then install the core:

> ./install.sh

Then run the core:

> ./run.sh
